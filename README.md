## Getting Started

1. Clone the repository:
```bash
git clone <repository-url>
cd dubai-space-travel
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.
